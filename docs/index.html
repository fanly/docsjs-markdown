@coding01/docsjs-markdown

Markdown Toolkit For docsjs

Convert docsjs HTML snapshots or DOCX into clean Markdown with customizable rules.

View on npm GitHub README Use with docsjs

Dual Entry Design

Use core HTML conversion from main entry, or DOCX conversion from /docx subpath.

Rule Plugins

Override conversion by HTML tag with custom rules for business-specific markdown styles.

CLI Included

Run direct conversion from terminal with frontmatter and parse-report options.

Recommended Pair: @coding01/docsjs

Import Word/DOCX with docsjs, then convert the HTML snapshot to Markdown with docsjs-markdown.

Quick Example

import { htmlToGfm } from "@coding01/docsjs-markdown";
import { docxToMarkdown } from "@coding01/docsjs-markdown/docx";

const md = htmlToGfm("<h1>Hello</h1>");
const docxMd = await docxToMarkdown(file, { format: "gfm" });
